Verizon In-Building End of Life Website

A site for Verizon's In-Building and Venue team that identifies sites with end-of-life (EOL) equipment, displaying relevant information for their replacement.

Features:

• A table view of information consolidated from Verizon’s vendors, Google Earth, and Verizon sources: associated vendors, DAS models, radio models, end of service dates, and end of life dates. Pulled data from live databases stored in Google Cloud Platform and an internal Verizon database.

• A search bar for user filtering.

• An embedded Tableau dashboard that displays the EOL IBV sites on a map of Verizon’s submarkets.

• A geographic map visualization of each site using OpenStreetMap’s API

• An easily accessible view of EOL site details to help local engineers & leaders determine whether sites should be decommissioned through consideration of factors like location and revenue.

Technology Stack:

• Frontend: HTML, CSS, Javascript

• Backend: PHP, Verizon Internal Databases, Google Cloud Platform

• Hosting: Verizon Internal Server
